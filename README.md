# Euri Ardila — Ingeniero Civil

Sitio web personal de Euri Ardila, Ingeniero Civil. Diseño estructural, gestión de obra e infraestructura.

## Tecnologías

- HTML5 semántico
- CSS3 puro (variables, grid, flexbox, media queries)
- JavaScript vanilla (Intersection Observer, smooth scroll)

## Cómo usar

Abre `index.html` directamente en cualquier navegador, o sirve los archivos con un servidor local:

```bash
python3 -m http.server 8080
```

## Características

- Navbar sticky siempre visible con efecto glassmorphism al scrollear
- Grid de proyectos asimétrico
- Tipografía dual: Cormorant Garamond (serif) + Inter (sans)
- Animaciones de entrada con Intersection Observer
- 100% responsive (mobile, tablet, desktop, wide desktop)
- Sin frameworks ni dependencias externas
